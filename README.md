A simple Python-based console application that allows users to book, cancel, and view doctor appointments while tracking available slots for each specialist. 
This project demonstrates core Python concepts and simulates a basic hospital appointment system.

**Features**
•	View available doctors and remaining appointment slots
•	Book an appointment with any specialist
•	Cancel a booked appointment
•	Prevents overbooking and invalid inputs
•	Case-insensitive doctor name matching
•	Menu-driven, user-friendly console interface

**Available Specialists**
•	General
•	Pediatrician
•	Gynacology
•	Cardialogy
•	Dermatology
Each doctor starts with 3 appointment slots by default.

**Concepts Used:**
•	Functions
•	Dictionaries
•	Lists
•	Conditional Logic
•	Loops
•	Exception Handling
•	Input Validation

**Workflow**
•	User enters a valid patient name
•	Chooses a specialist
•	System checks slot availability
•	Appointment is booked or declined based on availability
